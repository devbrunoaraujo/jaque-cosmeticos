# Plano: Módulo Fornecedores

## Resumo
Implementar CRUD completo de Fornecedores seguindo as mesmas convenções do módulo Produto.

## Entidade Fornecedor

| Campo | Tipo | Validação |
|---|---|---|
| id | Long (PK) | AUTO_INCREMENT |
| razaoSocial | String(150) | @NotBlank |
| nomeFantasia | String(150) | nullable |
| cnpj | String(14) | @NotBlank, unique, armazenar apenas dígitos |
| telefone | String(20) | nullable |
| email | String(150) | nullable |
| logradouro | String(150) | nullable |
| numero | String(10) | nullable |
| bairro | String(100) | nullable |
| cidade | String(100) | nullable |
| estado | String(2) | nullable |
| cep | String(8) | nullable, apenas dígitos |
| prazoEntregaDias | Integer | nullable |
| ativo | Boolean | default true |
| createdAt | LocalDateTime | @PrePersist |
| updatedAt | LocalDateTime | @PreUpdate |

## Arquivos a criar (8 arquivos)

### 1. Flyway Migration
`backend/src/main/resources/db/migration/V2__create_fornecedores.sql`
- Tabela `fornecedores` com snake_case, colunas espelhando a entity

### 2. Entity
`backend/src/main/java/com/seudominio/erp/fornecedor/entity/Fornecedor.java`
- @Getter/@Setter (sem @Data), @Entity, @Table(name = "fornecedores")
- @PrePersist / @PreUpdate para timestamps

### 3. Repository
`backend/src/main/java/com/seudominio/erp/fornecedor/repository/FornecedorRepository.java`
- extends JpaRepository<Fornecedor, Long>
- existsByCnpj(String cnpj)
- findByRazaoSocialContainingIgnoreCaseOrNomeFantasiaContainingIgnoreCase(String busca1, String busca2, Pageable)

### 4. DTOs
`backend/src/main/java/com/seudominio/erp/fornecedor/dto/FornecedorRequestDTO.java`
- record com @NotBlank, @Size, @Email validations

`backend/src/main/java/com/seudominio/erp/fornecedor/dto/FornecedorResponseDTO.java`
- record com factory method `of(Fornecedor)`

### 5. Service
`backend/src/main/java/com/seudominio/erp/fornecedor/service/FornecedorService.java`
- CRUD: criar, listar (com filtro por `busca` em razaoSocial OR nomeFantasia), buscarPorId, atualizar, desativar
- Validação de CNPJ único no criar e atualizar

### 6. Controller
`backend/src/main/java/com/seudominio/erp/fornecedor/controller/FornecedorController.java`
- @RestController @RequestMapping("/api/fornecedores")
- POST, GET `?busca=` (paged, busca em razaoSocial OR nomeFantasia), GET /{id}, PUT /{id}, DELETE /{id}

### 7. Teste de integração
`backend/src/test/java/com/seudominio/erp/fornecedor/FornecedorControllerIntegrationTest.java`
- @SpringBootTest + @ActiveProfiles("test") com H2
- Testes: criar, listar, buscarPorId
